To build SpeedKernel, you have to download and build
- zlib from http://www.zlib.net/zlib123.zip
- libpng from http://prdownloads.sourceforge.net/libpng/lpng1212.zip?download
- freetype2.2.1 from http://prdownloads.sourceforge.net/freetype/ft221.zip?download

Once built, put the required header files and libraries in your include directories and start building SpeedKernel.
